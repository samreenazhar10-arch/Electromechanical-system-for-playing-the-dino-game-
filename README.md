# Electromechanical System for Playing the Chrome Dino Game

## Overview
An automated system that plays the Chrome Dino Game by detecting obstacles on the screen using an LDR sensor and pressing the spacebar through a servo motor.

## Components Used
- Arduino Uno
- LDR (Light Dependent Resistor)
- Servo Motor
- Breadboard
- Connecting Wires
- Laptop or PC
- Mechanical Mount
- 
- ## Features
- Automatic obstacle detection
- Real-time response
- Servo-based key pressing mechanism
- Simple and low-cost design

## Working Principle
The LDR monitors changes in screen brightness caused by approaching obstacles in the Chrome Dino Game. When an obstacle is detected, the Arduino activates the servo motor, which mechanically presses the spacebar to make the dinosaur jump.
